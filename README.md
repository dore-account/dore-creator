# DORE

## フォルダ構成
```
DORE/
├── public/
├── src/
│   ├── components/  ...UIコンポーネントを配置する
│   ├── hooks/       ...汎用的なreact hooksユーティリティを配置する
│   ├── lib/         ...使用しているライブラリの固有のコード、初期化や設定のコードなどを配置する
│   ├── pages/       ...Next.jsでルーティングされるページを配置する
│   ├── styles/      ...全体のstyleを配置する
│   └── utils/       ...utilsロジックを配置する
└── types/
```
